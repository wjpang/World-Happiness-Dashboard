World Happiness Dashboard

This dashboard displays the world happiness by country over time, and explores the different metrics that could affect happiness.

World Happiness Score by Country per year

How has COVID-19 deaths affected happiness?

The Change in happiness displayed by this chart is the change from 2019 to 2020. Surprisingly, there is no correlation between the number of COVID-19 deaths and the change in countries' happiness.

How was happiness measured?

This chart displays the happiness as compared to other general happiness metrics. Since the happiness scores of each country was directly calculated with these metrics, there will always be a correlation between them.

This visualisation dashboard is created by Wen Juin Pang. The data sources are the 9th World Happiness Report Data Panel and Mortality Data. The report itself can be found here.
